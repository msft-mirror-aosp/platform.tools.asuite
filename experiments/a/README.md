# A tool

A tool is a command-line tool that can run android workflows and productivity tools
go/a-tool-design-doc

Contributions welcome!

### To Run
```a {config_name}```
or
```python3 a.py {config_name}```

### To develop
```python3 a.py {config_name}```

### To Test:
or
```python3 -m unittest **/*_test.py```
or
```python3 tests.py``
or
```atest .```
